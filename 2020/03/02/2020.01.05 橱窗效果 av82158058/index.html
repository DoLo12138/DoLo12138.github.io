2020.01.05 橱窗效果 av82158058

三月 02, 2020

步骤0 滤色

  • 打开 “人物” 图片

  • CTRL + J 复制图层
  • 导入 “背景” 图片

  • CTRL + T 变换背景大小
  • “混合模式” >> “滤色”

步骤1 色阶

  • 右下角 ○ >> “色阶”
  • 右下角最左边 “向下切入” 只对下方图层单独修改
  • 由于背景过亮 在最下方过渡带 将白块向左移动
  • 将上方的 “暗调” 和 “中间调” 向右移动

步骤2 高斯模糊

  • “滤镜” >> “模糊” >> “高斯模糊”

步骤3 混合选项

  • 右键 “背景” 图层 >> “混合选项”
  • 按住 “ALT” 键
  • 将 “下一图层” 的白色滑块的一半 向左移动
  • 使两个图层间的过渡平滑

步骤4 减淡工具

  • CTRL + SHIFT + ALT + E 盖印图层
  • 将 “人物” 图层的高光在 “背景” 中减淡 形成橱窗的反光
  • “减淡工具” >> “中间调” >> “曝光度：10%”
